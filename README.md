# mywhitespace

## How to use

please execute the line below

```python main.py path/to/your/file.ws```

- parser.py 
  -> parse through your file
- vm.py
  -> VM runs along your instructions
- main.py
  -> intergrate these 2 files above
- mkwsfile.ws
  -> make .ws program which outputs your message
